# prgroom CLI Foundation (8.1) Implementation Plan

> **For agentic workers:** Implement this plan task-by-task using the `test-driven-development` skill (red-green-refactor per task). Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Greenfield scaffold + core infrastructure for the `prgroom` Python CLI — the deterministic PR-grooming engine that replaces two agent skills. This bead is foundation only: state schema, Store Protocol + adapters, precondition/error tiers, escalation sink, config loader, deps seams, agent-contract Protocols, and wired-but-skeletal CLI verbs. No business logic.

**Architecture:** A fourth `uv`-managed Python package at `packages/prgroom/`, mirroring `installer`/`pdlc` toolchain verbatim. `src/prgroom/` layout. Cross-module dependencies sit behind `@runtime_checkable` Protocols; clock/randomness are injected via a `Deps` seam. Typer root with verb skeletons. State persists behind `prsession.Store` (file adapter for prod, in-memory adapter for tests).

**Tech Stack:** Python 3.11+, `typer` (runtime), `ruff`/`mypy --strict`/`pytest`+`pytest-cov`/`pip-audit` (dev). `uv` for env + console-script entry point.

## File Structure (package layout)

```
packages/prgroom/
  pyproject.toml              # uv package; typer dep; ruff/mypy/coverage stanzas mirror installer; fail_under=90, branch=true
  uv.lock                     # generated by `uv lock`
  src/prgroom/
    __init__.py               # version
    py.typed                  # PEP 561 marker (mirrors pdlc)
    cli.py                    # typer root + 12 verb skeletons + main() entry
    config.py                 # PrgroomConfig dataclass + TOML loader + duration parsing
    errors.py                 # Tier StrEnum, ErrorCode registry, PreconditionError, PrgroomError, exit_code_for_tier
    deps.py                   # Clock/Randomness Protocols + SystemClock/SystemRandomness + Deps bundle
    escalation.py             # Severity, Escalation, Sink Protocol, StderrSink, FileSink
    prsession/
      __init__.py
      pr_ref.py               # PRRef
      enums.py                # PRPhase, ItemKind, DispositionKind, ReviewerKind, ReviewerStatus
      state.py                # SCHEMA_VERSION, Identity, Disposition, ReviewItem, ReviewerState, QuiescenceState, PRGroomingState + to_dict/from_dict
      store.py                # Store Protocol, StateNotFoundError, SchemaUnknownError, StateCorruptError
      memory.py               # InMemoryStore (prod-importable per spec wording — but used by tests)
      file.py                 # FileStore + write_atomic + path resolver
    agent/
      __init__.py
      contracts.py            # CONTRACT_VERSION, ClusterContract/FixContract Protocols + input/output dataclasses
  tests/
    __init__.py
    conftest.py               # fixtures: tmp store dir, frozen clock, sample PRRef/state
    unit/
      __init__.py
      test_pr_ref.py
      test_enums.py
      test_state_serde.py
      test_store_protocol_fit.py
      test_memory_store.py
      test_errors.py
      test_escalation.py
      test_deps.py
      test_config.py
      test_contracts_fit.py
      test_cli_smoke.py
    integration/
      __init__.py
      test_file_store.py      # real filesystem round-trip + flock
```

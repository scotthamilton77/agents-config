# A.1 — uv-managed packages/installer/ Scaffold + CI Green — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Stand up `packages/installer/` as a uv-managed Python package with a thin `--help`-only CLI, top-level Make-based monorepo dispatch, and a single GitHub Actions workflow that runs the full quality gate (pytest + ruff + ruff-format + mypy --strict + pytest-cov 90% branch gate + pip-audit + actionlint + entry-binding verify).

**Architecture:** Total TDD — three smoke tests are written first and fail (red), then minimal `argparse`-based `main()` lands to turn them green. Static-analysis precedent locks in via `pyproject.toml` (broad ruff `select`, mypy strict + extra knobs, branch coverage 90% gate). Cross-package orchestration through a top-level `Makefile`; intra-Python-package commands through `uv run`. Single GitHub Actions workflow calls `make ci` so CI and local stay in lockstep.

**Tech Stack:** Python 3.11+; uv (package manager); pytest + pytest-xdist + pytest-cov; ruff (lint + format); mypy --strict; pip-audit; actionlint-py; GNU Make; GitHub Actions.

**Spec:** [`2026-05-17-w1qls.1.1-installer-scaffold-design.md`](./2026-05-17-w1qls.1.1-installer-scaffold-design.md)

**Bead:** `agents-config-w1qls.1.1` (parent chain already claimed `in_progress`)

**Worktree:** `.claude/worktrees/feat+w1qls.1.1-installer-scaffold/` on branch `worktree-feat+w1qls.1.1-installer-scaffold`. All commands below run from this worktree's root unless explicitly noted.

## Task 4: Top-level Makefile

**Goal:** Establish the monorepo dispatch precedent — single command surface for CI and local devs.

**Files:**
- Create: `Makefile` (at worktree root)

- [ ] **Step 1: Write `Makefile`**

Create `Makefile` at the worktree root with this exact content (Make requires literal TAB characters for recipe indentation, NOT spaces):

```make
.PHONY: ci ci-installer test-installer lint-installer format-check-installer \
        typecheck-installer cov-installer audit-installer lint-actions \
        verify-entry-installer

INSTALLER := packages/installer

ci: ci-installer lint-actions

ci-installer: lint-installer format-check-installer typecheck-installer \
              test-installer cov-installer audit-installer \
              verify-entry-installer

test-installer:
	cd $(INSTALLER) && uv run pytest -q

lint-installer:
	cd $(INSTALLER) && uv run ruff check

format-check-installer:
	cd $(INSTALLER) && uv run ruff format --check

typecheck-installer:
	cd $(INSTALLER) && uv run mypy --strict src

cov-installer:
	cd $(INSTALLER) && uv run pytest --cov --cov-report=term-missing

audit-installer:
	cd $(INSTALLER) && uv sync --frozen && uv run pip-audit

lint-actions:
	cd $(INSTALLER) && uv run actionlint

verify-entry-installer:
	uv --project $(INSTALLER) run python scripts/install.py --help > /dev/null
```

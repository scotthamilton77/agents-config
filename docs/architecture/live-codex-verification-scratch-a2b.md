# Live Codex Verification Scratch — A2b

This file exists to give scenario A2b (the review_summary ratchet) of a
live Codex-awareness verification exercise a real diff likely to draw a
genuine Codex comment. Cleanup of this file is automated by
`scripts/cleanup-live-verify-a2b.sh`, which runs nightly via cron and
deletes any scratch file older than 24 hours in this directory. Deleted
by a follow-up cleanup commit once the exercise's merge scenarios land.
